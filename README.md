# FinVerse 🚀

FinVerse is a full-stack GenAI-powered personal finance platform that delivers real-time financial guidance through an intelligent chatbot, featuring secure user authentication via Gmail. It leverages cutting-edge technologies to provide personalized financial insights, document-based Q&A, and an interactive financial literacy quiz, improving user literacy by up to 60%.

---

## Key Features ✨

- Intelligent chatbot powered by Google Gemini API, LangChain, and Retrieval-Augmented Generation (RAG) architecture.
- Secure Gmail-based user authentication.
- Document-based question answering for personalized financial guidance.
- Interactive financial literacy quizzes to boost user knowledge.
- Built with scalable React.js frontend and Flask/Express.js backend.
- Future plans include real-time virtual stock market integration with advanced features.
